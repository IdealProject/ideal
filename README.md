# Ideal, tu plataforma de aprendizaje libre

![Screenshot de la landing page de  ideal](./public/readme/ideal-shot.png)

Configurar para iniciar con el desarrollo

```bash
npm install -g pnpm
pnpm install
```

Para iniciar en local

```bash
pnpm run dev
```
